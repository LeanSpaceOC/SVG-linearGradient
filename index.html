SVG Icons, HTML

Gradient color and Hover CSS

Example of an flat design SVG icon with a hover css effect showing a gradient color. The SVG icon is taken from an icon library. A first svg HTML element is placed with an "ID" a the start of the body to declare the gradient color. A second svg HTML element is used to show the icon on the document. In the CSS file, the `ID` is used to charge the gradiant color when hovering the icon.
